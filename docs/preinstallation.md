# Предустановка необходимых программ и файлов

## Шаг 1. Установка программного обеспечения

Для дальнейшей настройки и работы с Обриком необходимо скачать и установить следующие программы и файлы:

* QGroundControl – программное обеспечение, необходимое для прошивки, настройки и калибровки полетного контроллера.

  Скачайте и установите версию для Windows/Linux/macOS с официального сайта [QGroundControl](https://docs.qgroundcontrol.com/master/en/qgc-user-guide/getting_started/download_and_install.html). При запросе **согласитесь на установку** дополнительных драйверов.
